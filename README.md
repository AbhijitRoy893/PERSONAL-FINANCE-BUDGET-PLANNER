# PERSONAL FINANCE & BUDGET PLANNER

## Overview

The **PERSONAL FINANCE & BUDGET PLANNER** is an Excel-based financial management project designed to help users efficiently manage their income, expenses, savings, and monthly budgets.

This project demonstrates the practical use of Microsoft Excel for financial analysis, data cleaning, budgeting, reporting, and dashboard visualization.

---

# Project Objectives

* Track monthly income and expenses
* Monitor personal budget performance
* Analyze spending patterns
* Create financial summaries using Pivot Tables
* Visualize data using interactive dashboards and charts
* Improve data organization and analysis skills using Excel
